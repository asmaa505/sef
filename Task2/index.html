SEF Academy Courses

SEF Academy is a professional training platform that specializes in teaching modern web development technologies. The academy aims to equip students with both the theoretical knowledge and practical skills needed to build complete web applications from scratch.

Fron End Development

In the front-end section, students learn how to design and create interactive and visually appealing websites. The following topics are covered:

  • HTML & HTML5
  • CSS & CSS3
  • Bootstrap
  • JavaScript (JS)
  • React.js

Back-End Development

In the back-end section, students dive into server-side programming and database management to make websites functional and data-driven. The main topics include:

  1. Node.js
  2. Express.js
  3. MongoDB
